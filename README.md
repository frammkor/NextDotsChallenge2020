# Challenge NextDots 2020 - RN 0.61.5 with Typescript

## Getting Started

`$ npm install`

## VSCode Extensions

- ESLint v2.0.11

## Launch Android

`$ npx react-native run-android`

## Launch on iOS

`$ npx react-native run-ios`


## Libraries Used

- React Native: v0.61.5
- React Native Dotenv
- React Native Vector Icons
- React Navigation v4
- Typescript
- Redux
- React Redux
- Redux Thunk
